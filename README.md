# AI Agent Templates

This repository contains a collection of templates for interacting with AI agents. These templates provide structure and guidance for AI-driven development tasks.

## Templates

- `templates/AGENT.md`: A template for providing instructions to an AI coding agent.
- `templates/RULES.md`: A template for defining development standards and rules for an AI agent to follow.
- `templates/PROJECT_README.md`: A comprehensive template for a project's `README.md` file.
